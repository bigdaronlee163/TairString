## 安装

### 1. 克隆代码
```shell
git clone https://github.com/wujunze/redis-module-panda.git
```

### 2. make

```shell
make
```
![file](https://lccdn.phphub.org/uploads/images/201808/02/4269/qRNCt1mHsv.png?imageView2/2/w/1240/h/0)
